Rover Base Station

The rover moves - so, it doesn't exactly have a static address. This page acts as a portal to the rover's server page, which is hosted on the rover itself. To save power, the rover will shut down after a period of inactivity, so most of the time the actual control server will not be reachable.

To tell the rover to wake up, click the button below. This will begin sending signals to the rover, and when it wakes up next, a response will be recieved and you can log into the control page. While the control page is open, the rover will stay awake.
