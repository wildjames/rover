Flask middleware is running

If you can see this, that means that the API scrubber is running. The controller sits behind this, and is not publicly accessible. It runs as root, which I don't want to directly expose, so this server takes input, checks it is of the correct form, cuts out the parts that are not needed, and then sends it to the controller. The controller then runs the hardware, and returns the result. The result is then sent back to the client, and the client is none the wiser.

Note that to access any of the API endpoints, you'll need to provide an auth token in the Authorization header of the request. That token can be found on the server, in the flask directory. If you can't access that, tough! Keep your paws off my rover!
